# rtems-dev-template
My RTEMS development tree template

I use this as a starting point for RTEMS development.

Directories:

tools
   - 4.11 - compiled toolchains for RTEMS 4.11 series
   - 5    - compiled toolchains for RTEMS 5 series
   - scripts - helper scripts for BSP configuration, RSB builds, etc
   - rtems-source-builder - I keep the RSB repository here 

rtems-src
   - This is where I checkout the RTEMS kernel repositories

apps
   - For RTEMS applications, demos, etc

bsps
   - 4.11 - RTEMS BSPs for 4.11 series installed here
   - 5    - RTEMS BSPs for 5 series installed here
